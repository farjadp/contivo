# Feature List

## Workspace Core

- Workspace creation with website URL
- Brand profile and growth engine tabs
- Activity logging and settings controls

## Brand & Strategy

- Brand Memory scrape + rescrape limit
- Brand Assets extraction and review flow
- Editable strategic sections (tone, value prop, audience, offers)

## Competitive Intelligence

- Competitor discovery with archive
- Competitor validation flow (accept/reject/manual)
- Market matrices generation (multi-chart)
- Competitor keyword analysis and strategy signals
- Products & Services comparison between client and competitors

## Content Generation

- AI ideation with framework selection (auto/manual)
- Required context enforcement: Brand Memory + Market Metric + Competitor Keywords
- Idea-to-pipeline handoff
- Draft preview generation
- Manual source injection (notes + attached files)
- Word-count targeting with per-platform limits

## Publishing Workflow

- Pipeline cards with generation and approval states
- Auto schedule defaults
- Quick publish schedule inline controls
- Calendar page with list/week modes (week chart placeholder)

## Strategic Reports

- AI-generated market intelligence reports (PDF + HTML)
- Monthly generation limit per user (5 reports/month)
- Eligibility gate: requires Brand Memory, 5+ Market Matrices, Competitor Keywords, and Products & Services data
- Gemini-powered HTML report designer with OpenAI fallback
- Puppeteer-based PDF rendering (print-quality, A4)
- Cover page, Executive Summary, Brand Profile, Competitive Landscape, scatter-plot Matrices, Keyword Intelligence, Products & Services, Strategic Recommendations
- Report history table with one-click PDF and HTML downloads
- Progress modal with 4-stage animated bar during generation

## Admin & Controls

- Gemini model control from admin panel
- Platform limits:
  - Competitive Landscape run limits
  - Brand Memory rescrape limits
  - Ideation max idea count
  - Word-count min/max per platform
- Framework metadata reporting
