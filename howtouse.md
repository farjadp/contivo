# How To Use

## 1) Create Workspace

- Go to `/growth`
- Create a new workspace with website URL
- Wait until brand analysis is ready

## 2) Prepare Intelligence Inputs

Before ideation, ensure these are available:

- Brand Memory
- Market Matrices
- Competitor Keywords

## 3) Generate Ideas

- Open workspace tab: `Ideation Station`
- Set goal/platform/funnel/framework mode
- Choose idea count
- Optionally set images and auto calendar insert
- Optionally set target word count
- Generate ideas

## 4) Move to Pipeline

- Use `Generate Preview` for draft check
- Click `Add to Pipeline`

## 5) Generate Final Draft

- In `Content Pipeline`, open a card
- Add optional manual notes/source files
- Set target word count
- Set quick publish date/time/timezone if needed
- Click generate

## 6) Publish Scheduling

- Use quick `Publish` or modal schedule
- After publish/schedule, app redirects to calendar page
- Validate scheduled items in `/growth/{workspaceId}/calendar`

## 7) Admin Controls

In `/admin`:

- change Gemini model
- update platform limits
- update ideation max content count
- update per-platform word-count min/max
